# Project 1: Alarm Clock

## Introduction

The objective of this project is to improve the timer mechanism in Pintos kernel. The current implementation of `timer_sleep` in `devices/timer.c` use the 'busy waiting' mechanism, where a sleeping process checks whether they should wake up or not at every timer tick. Thus, it should be improved by replace the 'busy waiting' design with a new block-based design. Code is available at [https://github.com/NohJanghan/pintos](https://github.com/NohJanghan/pintos.git).
